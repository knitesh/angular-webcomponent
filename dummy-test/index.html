Chatbot Web Component Demo

About this Demo

This is a simple HTML page demonstrating how to use the Angular Chatbot as a web component in any web application.

Implementation

To use the chatbot in your HTML page, you need to:

<script src="assets/runtime.js"></script>
<script src="assets/polyfills.js"></script>
<script src="assets/main.js"></script>
<app-chatbot></app-chatbot>

Customization

You can customize the chatbot using attributes:

<app-chatbot
  title="Custom Title"
  initial-message="Welcome to my custom chatbot!"
></app-chatbot>
